📘 Справка по модели

🎯 Что такое модель «ГрейнРус»?

Интерактивная бизнес-модель солодовенной компании. Позволяет изменять рыночные и управляемые параметры и мгновенно видеть их влияние на EBITDA, маржинальность, загрузку и другие KPI.

⚙️ Как управлять?

  • Слайдеры – меняйте значения, KPI пересчитываются автоматически.
  • Кнопки годов (2025–2028) – одинарный клик загружает последние настройки для этого года; двойной клик сбрасывает параметры к предустановленным (прогнозным) значениям.
  • Вкладка «Анализ + AI» – динамические рекомендации, редактор предустановок, AI-ассистент.

🧠 AI-ассистент

Пишите вопросы на естественном языке, например: «обнови цену ячменя на 2026 до 23500». AI изменит предустановки для указанных годов. После этого двойной клик по кнопке года применит новые значения. Сервер должен быть запущен (по умолчанию https://grainrus-ai.onrender.com/ask).

💾 Сохранение ситуаций

На вкладке «Анализ+AI» можно сохранить текущий набор параметров как ситуацию, а затем загрузить её двойным кликом или кнопкой. Ситуации хранятся в браузере.

✏️ Редактирование предустановок

В блоке «Редактирование предустановок годов» вы можете вручную изменить прогнозные значения для каждого года. После нажатия «Применить изменения» двойной клик по году будет использовать новые цифры.

📈 Графики

Три графика показывают тренды рыночных, управляемых и расчётных показателей на 2025–2028 гг. на основе текущих параметров как базового года.

🖨️ Печать

Кнопка «Печать» в заголовке выводит содержимое активной вкладки с сохранением графиков (преобразуются в изображения).

🚀 Запуск на другом компьютере

Достаточно открыть ссылку GitHub Pages. Сервер работает на Render, отдельной установки не требуется. AI-ассистент будет доступен с любого устройства.

📞 Вопросы и поддержка

При возникновении проблем проверьте консоль браузера (F12) и логи сервера на Render. Обращайтесь к разработчику.
